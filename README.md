# ⏱️ Pomodoro - Landing Page en Astro

Página informativa sobre la aplicación Pomodoro con análisis de productividad.

## 🚀 Quick Start

```bash
# 1. Instalar dependencias
npm install

# 2. Ejecutar servidor local
npm run dev

# 3. Abre en navegador: http://localhost:3000
```
